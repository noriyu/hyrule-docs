# hyrule-docs

## Overview
hyrule関連のドキュメントを管理するリポジトリ

ドキュメントページ: 
https://noriyu.github.io/hyrule-docs/

## 🐣 Getting Started 
ローカルサーバを起動する
```
npx docsify-cli serve docs
```
ローカルサーバにアクセスする
```
http://localhost:3000
```
